Completing the table¶

In order to draw the histogram, we need two more columns, namely the column with the classes'width and the column with the bar's heights.

In [20]:
fi = np.array(ztab['relative frequencies'][:])
lo = np.array(ztab['lo'])
hi = np.array(ztab['hi'])
In [21]:
width = hi-lo  # vector of classes' widths
height = fi/width # vector of classes' heights 
height
Out[21]:

We finally draw the histogram. Note that the first argument contains the lower bounds of the intervals while the second is the complete list of the bins.
