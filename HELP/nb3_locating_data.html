quantiles¶

We can define the quatiles vectorially. Here we define the quantiles $q_0=\min(x)$, $q_{0.25} = Q1$, $q_{0.5} = M$, $q_{0.75} = Q3$ and $q_1 =\max(x)$.

In [10]:
alpha=[0, 0.25,0.5,0.75, 1]
q = np.quantile(x,alpha, method = 'higher') # we discussed already the choice of the parameter higher during the lecture
q
Out[10]:
array([ 4810, 16607, 23116, 32428, 76407])

We can read graphically the quartiles on the eCDF:
